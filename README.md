# 🔢 Prime Number Checker (C++)

This project is a simple **C++ program that checks whether a given number is a Prime Number**.

It is a beginner-friendly program that demonstrates basic programming concepts such as **loops, functions, and conditional statements**.

---

## 📖 What is a Prime Number?

A **Prime Number** is a number greater than 1 that has only **two factors**:

* 1
* The number itself

### Example

Prime Numbers: 2, 3, 5, 7, 11
Not Prime: 4, 6, 8, 9

---

## ⚙️ How the Program Works

1. The user enters a number.
2. The program checks divisibility from **2 to √n**.
3. If the number is divisible by any value in that range, it is **not prime**.
4. Otherwise, it is a **prime number**.
